Introduction

Vcard is a personal digital business card template.

    This template is:
  • responsive (it will correctly resize depending on the screen you use; i.e: smartphone, tablet, desktop)
  • You can customize the content (text, photo, favicon*, Metadata*)
  • You can edit your content directly on the site using a username and password when you first register.

Before Installation

Here are a few things to be aware of before jumping into the installation.

Just like any other CMS such as wordpress, the vcard relies on creating a database where it stocks your content (text, photo, etc...). Most web hosting offer a database easy installation wizard that you can follow.

    You will also need to understand the directory of your web hosting and where you should upload the vcard files. Two methods can be used:
  • using your web hosting cPanel file manager*
  • using a FTP client*
Installation

Go to the following urland download the following files onto your computer:

- install-vcard.php - vcard_archive.zip

Just like any other CMS such as wordpress, the vcard relies on creating a database where it stocks your content (text, photo, etc...). Most web hosting offer a database easy installation wizard that you can follow.

Using either a CPanel or a FTP client, upload those 2 files in the root directory of your future website (usually holds the domain name you chose).

In your navigation bar, type the url of your root directory followed by "install-vcard.php". It should be something like

https://yourdomain.com/install-vcard.php

Follow the on screen instructions.

Once the vcard is installed correctly, go to the main page where you'll see shakespeare and click on "Made with a" link.

You can now log in. Click on the smily face in the footer. Edit your content.

Tips

Photos

It is important to use the right size for your photos. Your photos do not need to be gigantic in size (mb) to be picture clear on a web browser. More often than not, your photos for the web will not be printed.

Here are some useful links to learn in more depth about the right size for your photo.

https://www.1dogwoof.com/10-tips-best-image-size-blog/ https://www.calliaweb.co.uk/image-size-aspect-ratio-web-design/

Content

Try to keep your content short to engage your visitor. Long everlasting text will discourage the visitor.

Password

even though the vcard is secure, your password is only as secure as the one you choose. Try using a password manager. It will help you to create powerful passwords. Don't worry, if you can't remember your password one day, just use the forgot my password on the login page.
